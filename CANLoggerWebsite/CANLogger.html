NMFTA/NSF CAN Logger

The design and files associated with the secure controller area network (CAN) logger device and result sponsored by the National Motor Freight Traffic Association (NMFTA), Inc. and the National Science Foundation (NSF). This on-line resource links to the design details of the secure CAN logger and the code as infrastructure needed to enable the CAN data acquisition.

The resources on this page are released under an open source MIT license.

Login to View CAN Data Sets

The CAN Data Sets are stored in their encrypted form on Amazon's S3 system. Only registered users are allowed to have access to the data set keys.
